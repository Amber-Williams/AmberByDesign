Clinical Skin Tracker Mobile Application

I worked with a client to build out a clinical mobile application that would create consistency in result images for shown clinical efficacy. There was also an emphasis to have app-users complete questionnaires that backend-users would collect data from clinical studies on.

To create consistency with result images,I needed to create image overlays for the app-user to line up their body part with (such as a hand or face).


A information architecture was created so that the developers would understand the app page progression.

Next I drew wireframes that would serve as an outline of how users would interact with the app before the design was implemented.Following this I worked to create a clinical looking user interface that was on brand with the client.

Finally, a back-end needed to be created that the clinical physicians would use to collect app-user results and create new questionnaire studies for the app-users. Link to backend database prototype made with Adobe Experience Design

Once the prototype was completed and the developers understood the scope of the build out we worked to create the mobile application. Throughout the build out I worked to perform quality assurance testing on all features and functions. This ensured that the app-user would not run into issues on launch.

We soft launched the mobile application and it is currently in use for 3+ different clinical studies since its creation.
