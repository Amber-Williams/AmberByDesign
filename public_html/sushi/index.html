Client Customized Web Page

A Sushi restaurant wanted a new one page website that was on brand, Google Maps, easy to understand contact and hours section as well as consolidating a large menu that could be accessed effortlessly per subsection.

Outside of the clients wants, I noticed a need for Google Analytics to be implemented for the owners to track their customer demographics. Leading to pivoting towards different marketing strategies.

We started with a wireframe (insert wireframe picture) that the client could adjust, change elements as they pleased. The website was created to be fully responsive throughout any given device, screen or browser. After the wireframe was agreed upon. We moved to the design phase where the wireframe was decorated and vector images such as the title swords were created. (Design frame)

Moving forward to the development stage, the website framework was built utilizing Flexbox. Flexbox is a CSS built layout mode that is intended to accommodate different screen sizes. (Insert flex box image)

After development stage was completely built out, the client wanted to add a seprate section for their many delivery options as delivery was a major souce of revenue. Prior to the upload of this site, I took a back up from their current site incase the owners should ever want it.
